# Lakshay Chauhan — Portfolio

Personal data analyst portfolio site.

## Structure
- `index.html` — page markup
- `style.css` — all styling
- `script.js` — nav toggle, scroll-spy, scroll-reveal animations

## Run locally
Just open `index.html` in a browser, or serve with:
```
python3 -m http.server
```

## Deploy on GitHub Pages
1. Push this folder's contents to a GitHub repo (e.g. `lakshay-portfolio`).
2. Repo Settings → Pages → Source: `main` branch, `/ (root)`.
3. Add your `Lakshay_Chauhan_Resume.pdf` to the repo root so the Download Resume button works.
